THM-Pieces of a profile

Category OSINT
Difficulty Easy

Mission Objective

The investigation began with an isolated email address, cr0wseek_us3r@protonmail.com, to uncover the user’s hidden identity, location, and the final flag. Our goal was to follow the digital trail and piece together the profile.

Task 1: On which platform is this email registered?

To begin following the digital trail, our first step was to identify where the starting email, cr0wseek_us3r@protonmail.com, had been used to create accounts. This process is known as Reverse Email Lookup.

  • Tool Used: Holehe (a simple but effective Python tool that checks account registration via sign-up and password reset forms).

Task 2: What username is associated with this email?

Our investigation immediately pivoted to finding the user’s real online name, since the email’s name (cr0wseek_us3r) was likely a decoy. We started on X (Twitter), one of the platforms we confirmed in the first step.

We first searched for the exact name from the email, cr0wseek_us3r, but found nothing.
We then tried the simpler, core name, Crowseek. This successful search immediately led us to the target’s active Twitter account:

Task 3: Where does he live ?

To find the user’s location, we had to switch platforms because there were no clues on Twitter. Using the confirmed username, @cr0w5e3kn3t, we looked at the other accounts, which included Replit. We found the user’s Replit profile using the direct link, https://replit.com/@cr0w5e3kn3t(We could also use a tool like WhatsMyName to quickly find all of their social media pages).

Luckily, the user had written the city where they live right on their public Replit profile

Task 4 : On which platform does this username have the most public activity?

To find out where the user is most active, we checked their posts on the social media accounts we had already found. On the user’s Twitter profile (@cr0w5e3kn3t), we saw a post where they clearly stated: “I am really active on X!” Because the user told us directly that they are most active on this platform.

Task 5 : What is the final flag hidden in one of the accounts ?

Finding the final flag was the trickiest step because it was so well hidden. We checked Twitter and spent a lot of time trying to find the Spotify account, but the flag wasn’t in any of the normal spots :((

We focused on the most important clue: a weird post on the user’s Twitter that contained the code 5LbQULjs:

We realized this code was a unique ID that had to link to a hidden file somewhere else. Since the user was trying to hide a text secret, and Pastebin is the most common website for hiding text in these types of challenges, we logically decided to test the code as a Pastebin link. Our guess worked! We successfully went to https://pastebin.com/5LbQULjs, which immediately showed us the text file containing the final flag
